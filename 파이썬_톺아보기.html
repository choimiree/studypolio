파이썬¶

  • 프로그래밍 언어
    • 프로그래밍은 원하는 작업을 컴퓨터에게 시키는 것
    • 컴퓨터에게 내리는 명령어 형식이라고도 볼 수 있다.
  • 1991년 귀도 반 로섬이 만든 인터프리터 언어
  • 인터프리터 언어란 한 줄 씩 소스 코드를 해석해서 그때그때 실행해 결과를 바로 확인할 수 있는 언어이다.

변수 & 데이터타입¶

  • 변수
    • 숫자, 문자열, 리스트 등의 값을 가리키는 것(객체를 가리킨다.)
    • 값을 담아 놓는 공간이라고 생각할 수도 있다.
  • 데이터타입
    • 말 그대로 데이터의 타입(형식)을 의미
    • 파이썬에는 숫자(int, float, complex), 문자열(string), 불(bool), 컬렉션(list, dict, set, tuple), None 등이 있다.

조건문 & 반복문¶

  • 조건문(if)
    • 조건에 따라 프로그램의 흐름을 제어하기위한 구문
    • 파이썬에서는 if / elif / else 형식으로 표현
  • 반복문(for/while loop)
    • 특정 반복조건에 따라 코드 블록을 반복시키기 위한 구문

함수¶

  • 특정 코드를 재사용할 수 있도록 구성한 코드 형태
  • 인풋과 아웃풋이 있다

컬렉션¶

  • 컬렉션 데이터 타입들은 기본적으로 여러 값을 묶어서 사용하기 위한 형태
    • 리스트: 변경 가능하며 인덱싱이 가능(순서가 있다.) []
    • 딕셔너리: 변경 가능하며 key-value형태 {}
    • 튜플: 리스트와 비슷하지만 변경 불가능(immutable) ()
    • 셋: 키가 없는 딕셔너리 형태, 순서가 없다, 집합을 나타내기 위한 형태

클래스¶

  • (객체지향 프로그래밍에서 상태(값-변수)와 행위(처리-함수)를 하나의 묶음 형태로 사용하는) 객체의 설계도
  • 클래스의 실체가 바로 객체

  • 클래스에 의해 만들어진 객체 == 인스턴스(실제 메모리에 올라가 있음)

    쿠키틀 == 클래스/쿠키 == 객체/쿠키 == 쿠키틀의 인스턴스 즉, 쿠키틀이라는 클래스와의 관계를 내포하여 말할 때에는 인스턴스

모듈¶

Lambda¶

  • 람다 표현식은 익명 함수을 만드는 방식
  • 익명 함수는 말 그대로 함수 이름이 없는 함수
  • 람다 표현식으로 만든 익명 함수는 변수에 할당하여 호출할 수 있음

def lambda(parameters):

return expression

def find_max(a):
    n = len(a)
    max_value = a[0]
    for i in range(1,n):
        if a[i] > max_value:
            max_value = a[i]
    
    return max_value

num_list = [12,1,5,28,20,46,36,21]
print(find_max(num_list))

#동명이인 (같은 값)을 찾아 셋(set)으로 반환하는 함수
def find_same_name(a):
    n = len(a)
    result = set()
    for i in range(0,n-1):
        for j in range(i+1, n):
            #print(f"i={i} / j = {j}")
            if a[i] == a[j]: #이름이 같으면
                result.add(a[i]) #찾는 이름을 result에 추가
                
    return result

names = ["Tom","Jerry","Ethan","Tom"]
find_same_name(names)
Out[76]:
{'Tom'}

#딕셔너리를 이용하여 동명이인 (중복된 값)을 찾는 함수
def find_same_name_using_dict(a):
    #1단계: 각 이름이 등장한 횟수를 딕셔너리로 만듦
    name_dict = {}
    for name in a:
        if name in name_dict: #이름이 name_dict에 이미 있으면
            name_dict[name] += 1 #등장횟수를 1증가 시킴
        else: #name_dict에 아직 해당 이름이 없으므로
            name_dict[name] = 1 #해당 이름을 키로하고 값을 1로 하여 저장(등장횟수:1)
            
    #2단계: 만들어진 딕셔너리에 등장횟수가 2이상인 것을 찾아 결과 셋에 추가
    result = set()
    for name in name_dict:
        if name_dict[name] >= 2:
            result.add(name)
    return result

names = ["Tom","Jerry","Ethan","Tom"]
find_same_name_using_dict(names)
Out[84]:
{'Tom'}

재귀호출(recursion)이란?¶

재귀 호출은 함수가 자기 자신을 다시 호출하는 것

#1부터 n까지의 합을 구하는 문제를 이용하여 간단히 팩토리얼 구하기
def factorial(n):
    result = 1
    for i in range(1,n+1):
        result *= i
    return result

print(factorial(3))

#재귀호출을 이용하여 팩토리얼 구현하기
def factorial(n):
    if n <= 1:
        return 1
    return factorial(n-1)*n

print(factorial(3))

6

피보나치 수열¶

  • 피보나치 수열은 첫째 및 둘째 항이 1이며 그 뒤의 모든 항이 바로 앞의 두 항의 합인 수열이다. 편의상 0번째 항을 0으로 두기도 한다.
  • 피보나치 수열 역시 재귀 호출을 설명할 때 자주 등장하는 예시
In [68]:
#재귀 호출을 이용하여 피보나치 수열의 n번째 항 구하기
def fibo(n):
    if n <= 1:
        return 1
    return fibo(n-1) + fibo(n-2)

for i in range(0,10):
    print(fibo(i))
